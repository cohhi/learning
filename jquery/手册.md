# jquery查询手册

- 效果

```js
hide()          //隐藏
show()          //显示
toggle()        //隐藏/显示

fadeIn()        //淡入
fadeOut()       //淡出
fadeTo()        //设置透明的0-1之间
fadeToggle()    //淡入/淡出

slideDown()     //下滑动
slideUp()       //上滑动
slidToggle()    //上/下滑动

animate()       //自定义动画

stop()          //停止方法

speed()         //速度
normal          //正常速度
fast            //快速
slow            //慢速
1000            //毫秒单位
```



- DOM

```js
//捕获
html()			//返回内容的
text()			//返回文本的
val()			//返回表单字段的值

//设置
html()			//设置内容
text()			//设置文本
val()			//设置表单字段
attr()			//回调函数

//添加元素
append()		//末尾添加
prepend()		//开头添加
after()			//元素后添加
before()		//元素前添加

//删除元素
remove()		//删除元素及子元素
empty()			//删除元素的子元素

css类
addClass
removeClass
toggleClass
CSS
```
