# jquery查询手册

- 效果

```js
hide()          //隐藏
show()          //显示
toggle()        //隐藏/显示

fadeIn()        //淡入
fadeOut()       //淡出
fadeTo()        //设置透明的0-1之间
fadeToggle()    //淡入/淡出

slideDown()     //下滑动
slideUp()       //上滑动
slidToggle()    //上/下滑动

animate()       //自定义动画

stop()          //停止方法

speed()         //速度
normal          //正常速度
fast            //快速
slow            //慢速
1000            //毫秒单位

```
